HTML

HTML (Lenguaje de Marcas de Hipertexto , del ingles HyperText Markup Lenguage) es el componente mas basico de la web. Define el significado y la escritura del contenido web. Ademas de HTML , generalmente se ultilizan otras tecnologias para describir la apariencia/presentacion de una pagina web (css) o la funcionalidad/comportamiento (JavaScript).
